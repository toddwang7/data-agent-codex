# ad-analysis config

This directory contains the first structured configuration draft for the
advertising data analysis agent.

Files:

- `agent.json`: agent-level definition
- `semantic-config.json`: field semantics, rules, metrics, and clarifications
- `report-templates/single-month-report.json`: single-month monthly report template
- `report-templates/multi-month-report.json`: cross-month monthly report template

This is a product-facing configuration draft. It is intended to show what the
future platform should be able to configure, not just what the runtime will
store internally.
